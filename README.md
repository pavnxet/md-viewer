# Markdown Viewer

A lightweight, single-file Markdown viewer with live preview. Drag and drop a `.md` file or paste markdown to view it instantly.

**Live:** [https://pavneet-singh.github.io/md-viewer/](https://pavneet-singh.github.io/md-viewer/)

## Features

- Drag-and-drop or paste markdown content
- Live split-pane editor + preview
- Dark/light theme toggle
- Font picker (System, Inter, Noto Sans, Fira Code, JetBrains Mono)
- Synchronized scrolling
- Code syntax highlighting via highlight.js
- Print support
- Mobile responsive
- Remembers theme and font preferences (localStorage)

## Usage

Open `index.html` in a browser, then drag a `.md` file onto the page or paste markdown content.
